# Дополнительные услуги для авиапассажиров

MVP сервиса выбора дополнительных услуг при покупке авиабилетов.

## Запуск

```bash
# Установка зависимостей
pnpm install

# Запуск dev-сервера
pnpm dev
```

## Сборка

```bash
pnpm build
```

## Линтинг

```bash
# Проверка
pnpm lint

# Форматирование
pnpm format

# Проверка + автоисправление
pnpm check
```

## Стек

- React 19
- TypeScript
- Vite
- Styled Components
- Biome
